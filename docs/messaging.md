Guidelines for consistent communication across all VelocityPulse marketing materials.

## Voice & Tone

### Voice Attributes

| Attribute | We Are | We Are Not |
|-----------|--------|------------|
| **Confident** | Assured, clear, direct | Arrogant, dismissive |
| **Helpful** | Supportive, educational | Condescending, preachy |
| **Approachable** | Friendly, human, accessible | Overly casual, unprofessional |
| **Honest** | Transparent, realistic | Exaggerating, misleading |

### Tone Guidelines

**Default tone:** Confident and helpful

- **Homepage:** Confident, exciting, clear value proposition
- **Documentation:** Helpful, patient, thorough
- **Error messages:** Apologetic, solution-focused
- **Pricing page:** Transparent, fair, no pressure
- **Support:** Empathetic, efficient, human

### Writing Style

1. **Short sentences.** Easy to scan.
2. **Active voice.** "VelocityPulse discovers devices" not "Devices are discovered"
3. **Plain English.** No jargon unless necessary.
4. **Second person.** "You" and "Your" over "Users" and "Customers"
5. **No fluff.** Every word earns its place.

## Words to Use

|------------|-----|
| Monitor | Watch, track, see |
| Infrastructure | Network, devices, systems |
| Deploy | Install, set up |
| Configure | Set up, customize |
| Endpoint | Device |
| Solution | Tool, product, platform |
| Leverage | Use |
| Utilize | Use |
| Functionality | Features |

## Words to Avoid

| Word | Why |
|------|-----|
| Revolutionary | Overused, meaningless |
| Best-in-class | Vague, unproven |
| Synergy | Corporate jargon |
| Disruptive | Overused |
| AI-powered | Unless specifically true |
| Enterprise-grade | We're the anti-enterprise on pricing |
| Robust | Vague |
| Scalable | Everyone says this; say "unlimited" instead |
| Next-generation | Meaningless |
| Cheap | Use "affordable" or specific price instead |

## Formatting Guidelines

### Numbers
- Spell out one through nine
- Use numerals for 10 and above
- Always use numerals for: prices ($50), percentages (50%), time (10 minutes)

### Capitalization
- Sentence case for headlines
- Title Case for product names only
- Never ALL CAPS except acronyms

### Punctuation
- Oxford comma (a, b, and c)
- One space after periods
- No exclamation marks in body copy (one per page max in marketing)

### Product References
- First mention: VelocityPulse
- Subsequent: VelocityPulse or "the platform"
- Never: "the product," "the solution"
