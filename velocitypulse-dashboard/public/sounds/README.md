# Sound Assets

## sonar-ping.mp3

This directory should contain a `sonar-ping.mp3` file for the ping/pong feature.

You can download a free submarine sonar ping sound from:
- https://freesound.org/search/?q=sonar+ping
- https://www.soundsnap.com/tags/sonar
- https://mixkit.co/free-sound-effects/ping/

Requirements:
- Duration: ~0.5-1 second
- Format: MP3
- Recommended: Submarine sonar ping or radar blip sound
